What is DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

  • document.querySelectorAll()
what is event delegation in the context of the dom,and why is useful?

Event Delegation is a technique in JavaScript that allows you to manage events more efficiently by using a single event listener at a higher level in the DOM tree instead of adding event listeners to individual elements. This is particularly useful when dealing with dynamically added elements or when you have a large number of similar elements.
